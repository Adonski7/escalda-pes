# Verde Essence

Projeto corrigido em React + Vite + TypeScript.

## Como rodar

1. Extraia o ZIP.
2. Abra a pasta no VS Code.
3. No terminal, rode:

npm install

npm run dev

4. Abra o link que aparecer no terminal.

## Onde editar

- Textos e produtos: `src/App.tsx`
- Visual/cores/tamanhos: `src/App.css`
